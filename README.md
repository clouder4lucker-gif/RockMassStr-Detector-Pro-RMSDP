# RockMassStr-Detector-Pro-RMSDP
A fast and open-source software for automated detection of rock joints

Software installation process

1. Create an empty folder. （You can name it whatever you like.）
2. Enter the folder you created and then create another folder named "_internal".
3. Download the branch "Installation Package 1".
4.Locate the file named "RockmassStrDetPro.exe" in the downloaded branch (installation package 1), and move it separately to a folder at the same level as "_internal"; all the other files should be placed uniformly in the "_internal" folder.
5. Enter the "_internal" folder and Create an empty folder named "tensorflow.
6. Download the branch "Installation Package 2"
7. Move the recently downloaded branch (Installation Package 2) into the newly created folder (tensorflow) and rename the folder named "python1" to "python".
8. Download the branch "Installation Package 3" and put it in the folder whose name has just been changed to "python".
installation complete!!!!

Note：
    1. Users only need to click on the "RockmassStrDetPro.exe" file to use it.
    2. All ".gitattributes" files can be deleted altogether.
